关于gridea-theme-pure-mod

· 2020-02-10 · # Gridea # 主题

gridea-theme-pure的魔改版，也就是我的博客现在使用的主题

项目地址：🚀Github

介绍

本主题魔改自🚀gridea-theme-pure

本主题是一款专为 Gridea 设计的主题，默认内置 "谷歌蓝" 配色方案。另外还：

  • 支持自定义配色。
  • 移动端页面适配。
  • 支持社交信息设置。
  • 支持谷歌统计。

安装

使用 git clone 本项目到 Gridea 站点源文件路径的 themes 目录内，然后重启 Gridea 即可。

Gridea 站点源文件路径：可在 Gridea 客户端设置页面查看。
